# 💨 Tailwind CSS Cheat Sheet Interativo

Bem-vindo ao **Tailwind Fácil**! O Tailwind CSS é incrível, mas com milhares de classes utilitárias disponíveis, é fácil esquecer algumas delas. Este projeto é um guia rápido e interativo que reúne os **99% do feijão com arroz** que usamos no dia a dia do desenvolvimento Front-end.

## 🎯 Objetivo do Projeto
Oferecer uma ferramenta de **consulta ultrarrápida** para estudantes e desenvolvedores, permitindo encontrar a classe exata para margens, flexbox, grid, cores e responsividade sem precisar navegar pela documentação inteira.

## ✨ Funcionalidades e UI/UX
- **Design Premium:** Interface moderna em *Dark Mode* com efeitos de *Glassmorphism* (fundo desfocado e semi-transparente).
- **Busca em Tempo Real:** Filtra instantaneamente pelo nome da classe, propriedade ou descrição.
- **Cópia Inteligente:** Botão de copiar que aparece suavemente no *hover*, com um pop-up (Toast) de confirmação elegante.
- **Categorização Lógica:** De Espaçamentos e Flexbox até Efeitos Visuais e Modificadores de Estado (`hover:`, `dark:`).

## 🛠️ Tecnologias Utilizadas
- **HTML5** (Estrutura semântica)
- **Tailwind CSS** (Estilização via CDN)
- **Vanilla JavaScript** (Lógica de busca e *clipboard*)
- **Phosphor Icons** (Ícones limpos e modernos)
- **Google Fonts** (Fonte *Inter* para UI e *Fira Code* para os blocos de código)
